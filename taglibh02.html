@NavChild

@NavChild 标签

调用语法

<@NavChild items="navList"></@NavChild>

模板示例

<ul class="nav navbar" role="navigation">
    <@NavChild items="navList">
    <#list navList as it>
        <li>
            <a href="${it.url!}" class="<#if it.active >active</#if>">${it.name!} </a>
        </li>
    </#list>
    </@NavChild>
</ul>

数据结构

NavItem 数据结构如下：

字段 数据类型 说明
id Integer ID
pid Integer 父级ID
parent NavItem 父级
sort Integer 排序
name String 栏目名称
url String URL地址
child List NavItem 子栏目集合
active boolean 是否为当前栏目
end boolean 是否是结束标记
